# RobotMatrix — Unity 机械臂运动学与控制系统

[![Unity](https://img.shields.io/badge/Unity-2021.3+-black.svg)](https://unity.com)
[![C#](https://img.shields.io/badge/C%23-9.0-blue.svg)](https://docs.microsoft.com/dotnet/csharp/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

> **高精度 N-DOF 机械臂运动学与控制系统** — 纯 C# 实现，零第三方依赖

---

## 功能特性

- **正运动学 (FK)** — DH 参数自动计算，链式矩阵相乘
- **逆运动学 (IK)** — DLS 阻尼最小二乘法，支持位置+姿态约束
- **N-DOF 通用** — 不限关节数量，支持旋转/移动关节
- **关节限位** — 软硬限位保护，限位触发可视化提示
- **平滑插值** — 关节角度速度限制 (180°/s)，防止跳变
- **多坐标系支持** — World / Base / TCP 增量控制
- **可视化调试** — Scene 视图 Gizmo 实时显示坐标轴、连杆、DH 坐标帧
- **交互控制** — 键盘控制 (FK/IK)
- **异步计算** — 后台线程 IK 求解，不阻塞主线程
- **引擎无关核心** — 数学库与运动学层可移植到其他引擎

## 快速开始

### 安装

1. 将 `RobotMatrix` 文件夹复制到 Unity 项目的 `Assets/` 目录
2. 在场景中构建机械臂关节层级（父子或独立节点均可）
3. 在根物体上添加 **RobotMatrix > Robot Arm** 组件
4. 配置关节 Transform 列表和参数，点击 Play 即可控制

### 最小配置

| 必填项 | 说明 |
|--------|------|
| `jointTransforms` | 按运动学链序排列的关节 Transform |
| `axisDirection` | 各关节的局部旋转轴方向 |

其余参数均有合理默认值。

> **⚠️ 重要提示**：末端执行器 (TCP) 必须挂载在最后一个关节的子物体上，否则 IK 计算会出现偏差。

## 控制方式

### 键盘控制

| 模式 | 按键 | 功能 |
|------|------|------|
| 通用 | `Tab` | 切换 FK / IK 模式 |
| FK | `Q` / `E` | 选择上一个/下一个关节 |
| FK | `A` / `D` | 负向/正向旋转选中关节 |
| IK | `W` `S` `A` `D` | Z轴/前后、X轴/左右移动 |
| IK | `Space` / `Ctrl` | Y轴/上下移动 |
| IK | `I` `K` `J` `L` `U` `O` | 绕 X/Y/Z 轴旋转 |

## 核心算法

| 算法 | 实现 | 说明 |
|------|------|------|
| FK | DH 矩阵链式相乘 | O(DOF) 复杂度 |
| IK | DLS 阻尼最小二乘 | `Δθ = J^T (JJ^T + λ²I)^(-1) e` |
| 雅可比 | 几何法 | 6×N 矩阵，支持旋转/移动关节 |
| 插值 | 速度限制 | 最大 180°/s，帧率无关 |

## 技术规格

| 特性 | 参数 |
|------|------|
| 关节数量 | 无限制 (N-DOF) |
| 关节类型 | 旋转 (Revolute) / 移动 (Prismatic) |
| IK 迭代 | 默认 50 次，最大 200 次 |
| 位置精度 | 0.001 m (1mm) |
| 旋转精度 | 0.001 rad (~0.057°) |
| 求解耗时 | 6-DOF 约 5~10ms (同步模式) |
